# Calculator App

A simple web-based calculator that adds three numbers together.

## Description

This is a lightweight HTML calculator application that allows users to input three numbers and calculate their sum. The calculator features a clean, modern interface and runs entirely in the browser using vanilla JavaScript.

## Features

- Add three numbers together
- Clean and modern user interface
- Input validation
- Responsive design
- No external dependencies required

## How to Use

1. Open `index.html` in any modern web browser
2. Enter three numbers in the input fields
3. Click the "Add" button
4. View the result displayed below the button

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
